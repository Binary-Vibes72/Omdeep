## Getting Started
### Prerequisites
- Code editor (Recomended: Visual Studio Code)
- Git installed on your local machine

### Installation
1. Clone the repository
```
git clone https://github.com/yourusername/omdeep-enterprises-website.git
```

2. Navigate to the project directory

```
cd omdeep-enterprises-website
```

### Install Visual Studio Code Extension:

Live Server by Ritwick Dey:
